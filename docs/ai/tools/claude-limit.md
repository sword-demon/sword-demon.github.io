# Claude 使用注意

## 怎么尽可能避免被封

尽量固定IP，可以长期稳定使用

1. 新账号前两周，只用 Claude Desktop里的Claude Code，不碰 CLI 

2. 新账号阶段，不使用 OpenCode/OpenClaw/CraftAgent等第三方客户端

3. 通过App Store或Google Play 订阅，不在官网直接刷卡

4. 从$20 Pro Plan起步，多次触碰限额后再自然升级

5. 升级Max Plan后前两周，触发用量上限不超过3次

6. 账号要有作息一一不要24小时持续高负载

7. IP类型无所谓，但别用万人骑的公共代理，且早期不要频繁切换IP

8. 被封过号的，换账号前彻底清除电脑上一切Claude相关文件(指纹重置)

9. 不要共享账号给任何人

10. 养号期约一个月(首次续费后)，之后上述限制可以有序放松
