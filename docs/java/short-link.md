# 短链平台技术栈

## 微服务技术

- 新版`Spring Cloud Alibaba`全家桶
- `SpringBoot 2.5` + `Ncos2.x`+`MyBatisPlus`



## 分布式

- 分布式缓存
- MQ 中间件
- 分布式调度



## 海量数据

- `MySQL8.0` + `ShardingSphere`多维度分库分表
- 阿里云`OSS` + `ElasticSearch7.x`



## 实时计算+数据处理+存储可视化

- `Flink1.13`
- ClickHouse
- HDFS
- 数据清洗分层
- `ECharts`可视化数据

## 分布式链路追踪+监控+持久化存储

- Apache Skywalking
- 监控报警推送
- ElasticSearch7.x 持久化

## DevOps 全链路部署+生产环境流量压测

- Jenkins CICD
- 阿里云 Git 仓库
- 阿里云 ECS CentOS7
- Rancher2.5
- 容器编排调度
- Jemeter5.x 单接口压测
- 混合流量录制
- 流量压测扩大重放
